Architecture, optimization, and maintenance

Key points of the project:

  • For this large customer in the Clean Energy area, this contract is intended for architectural restructuring, performance improvement of APIs , and Migration of Framework (.NET Framework 4.8 for .NET Core). Some points of improvement made in legacy systems were the use of good practices in Entity Framework 6, improvement in the use of queries, Lambda functions, and LINQ to SQL.
  • The migration strategy adopted began with uncoupling legacy code to specific libraries, replacing them with consumption with Rests APIs, and removing direct access to the database.
