Sales Success Planner

Key points of the project:

  • We delivered a complete solution to the life insurance sales team; even with time challenges and product complexity, I came in to contribute as an expert in hybrid mobile development with Ionic Framework.
  • With the product's success, it was adopted by the entire sales force in Brazil and became a model for the headquarters in the USA.
  • We use a variety of mobile device native API features to meet area demands, such as:
    • Phonebook integration for better seller customer management.
    • Push Notifications for assistance in scheduled meeting reminders.
    • Handling App Settings and LocalStorage to keep the user's settings saved.
    • Offline mode is one of the main needs requested by users due to regions that do not have easy access to the Internet and that even then, it was possible to start a sale.
    • Due to the nature of sensitive data, we use a tool for encryption and data security and a barrier against access to data files compiled via .ipa (iOS) and .apk (Android) with the company's Mobile Iron solution.
  • The programming stack involved Angular 2+/Typescript, Ionic Framework 2+/Typescript , WebApi C#/.NET, SQL SERVER relational database
  • In this same project, we implemented the Token Management Service (TMS) for credit cards according to PCI DSS, with the aid of APIs of the company Cybersource, created an API in WCF/C# used by the whole company in this process of security and compliance.
